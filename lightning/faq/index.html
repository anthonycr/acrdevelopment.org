FAQ

What is the difference between Lightning and Lightning+?

Lightning+ is the same as Lightning, except that it disables the 10 tab limit and allows you to use ad block. You can purchase Lightning+ on the Play Store or if you can't or don't want to do that, you can download the latest APK for free from F-Droid.

Why is my search bar colored on different websites?

This is a feature called color mode. It takes the favicon of the website and tries to guess a color that would match the website. Sometimes it's wrong. If you want to turn it off go to General Settings and change the "Color Mode" setting.

Can I change the way tabs are displayed?

Yes, tabs can be shown either in sliding navigation drawer (default on phones), or they can be shown like they are on a desktop browser in a tray at the top of the screen (default on tablets). If you want to change it, then go to General Settings and change the "Show tabs in Navigation Drawer" option to your desired setting.

How do I exit the browser or close all tabs?

The easist way is to long-press on the back button. This brings up a menu that allows you to close the current tab, or all the tabs, or all the other tabs. Also, if you're using the tab drawer UI option, the three line icon in the top left hand corner of the tab drawer will show the same menu when clicked.

Why is Adobe Flash grayed out in my settings?

Adobe dropped support for flash a few years ago on Android, and consequently Android dropped support for the flash plugin. If you are on a version below Android 4.4 (Jelly Bean and ICS) then you can use Flash, you will need to install the Flash APK. If you are on Kitkat or above you cannot use Flash. Sorry :(

Why is Block Ads grayed out in my settings?

Block ads is a Lightning+ feature, see the first question for download links.

Why am I limited to 10 tabs?

This is a limitation of the free version of Lightning, Lightning+ allows unlimited tabs, see the first question for download links.

Can I import bookmarks from another browser?

Up until recently, you could import bookmarks from Chrome or the stock browser. However, Google seems to have removed the functionality so it's not guaranteed to work anymore.

Why is Request 'Do Not Track' grayed out in settings?

Unfortunately, only Android version kitkat 4.4 and up support this feature. If you're on a lower version, it won't work because it breaks page loading.

Why is Remove Identifying Headers grayed out in settings?

Unfortunately, only Android version kitkat 4.4 and up support this feature. If you're on a lower version, it won't work because it breaks page loading.

Why is Block 3rd Party Cookies grayed out in settings?

This is a new feature to the WebView and is only supported on Android version Lollipop 5.0 and up.

How can I allow sites like Google maps to access my location?

Go to Privacy Settings and enable location access, then when you navigate to the site, you will be prompted to allow the site to access your location.

Why doesn't the Save your passwords setting save your passwords?

Google unfortunately removed this specific functionality from the rendering engine that Lightning uses. The setting also controls saving form data which means when you enter data into a form, the WebView can remember what you put in the last time... just not for passwords.

Why is the keyboard covering the text box in the webpage?

If this is happening, you probably have the "Hide status bar" option enabled. The problem is that with that setting enabled, Android decides that the app shouldn't get a chance to resize if the keyboard shows up, and so there is no way to know when the keyboard is displayed with that setting enabled. Unfortunately, this is the way they want the option to work so there isn't anything I can do about it. If it's really bothersome, I suggest you turn the setting off and then the keyboard won't be hiding the text box.

What rendering engine does Lightning use?

Lightning Browser uses the built int WebView renderer included on the Android platform. On newer devices this is an app that updates on your device called "Android System WebView."
